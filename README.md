# Fantabtic

Turn your boring new tab page into an inspirational beautiful page.

## Features

- Daily wallpaper
- Daily quote
- Top visited sites
- View more sites button
- Date and time
- App list and web store
- Settings to turn off/on all features
- Synced settings and storage

## Libraries

- AngularJS
- Bootstrap 4
- jQuery
- Aninate.css
- Bing (Wallpaper)
- Quotes.rest (Quotes)
- Popper.js (For bootstrap)
